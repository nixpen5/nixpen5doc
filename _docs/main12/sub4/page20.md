<pre>
  <t2><bold>✔ 국세청 연말정산간소화 의료[요양]기관 자료제출 방법</bold></t2>
    ✎ https://www.hometax.go.kr 접속 후 회원 가입 또는 로그인
       ( 공인 인증서가 등록되어있지 않은 경우 등록 필요함.)
</pre>

[![](/images/{{page.url}}_1.png)](/images/{{page.url}}_1.png){: target="_blank"}
<pre>
<red>※ 국세청 자료전송 관련 문의 사항은 국세청 연말정산간소화 고객센터에 문의 바랍니다.</red>
    <blue>문의처 : 국번없이 126</blue>
</pre>

<pre>

    ✎ 홈택스 메인화면 상단 메뉴에서 ‘조회/발급’ 탭에서 ‘연말정산간소화’ 클릭
</pre>
[![](/images/{{page.url}}_2.png)](/images/{{page.url}}_2.png){: target="_blank"}


<pre>

    ✎ 화면 하단의 ‘소득·세액공제 자료’ 제출 클릭
</pre>
[![](/images/{{page.url}}_3.png)](/images/{{page.url}}_3.png){: target="_blank"}

<pre>

    ✎ 제출 자료종류를 ‘의료비’로 선택하고, ‘자료제출’ 버튼 클릭
</pre>
[![](/images/{{page.url}}_4.png)](/images/{{page.url}}_4.png){: target="_blank"}

<pre>

    ✎ 파일 선택 -> 파일 찾기 -> 선택 -> 업로드 버튼 클릭
</pre>
[![](/images/{{page.url}}_5.png)](/images/{{page.url}}_5.png){: target="_blank"}

<pre>

    ✎ 제출 파일 검증하기 완료 후 제출 대상 체크하여 ‘제출하기’ 버튼 클릭
</pre>
[![](/images/{{page.url}}_6.png)](/images/{{page.url}}_6.png){: target="_blank"}

<pre>

    ✎ 제출이 완료된 화면
</pre>
[![](/images/{{page.url}}_7.png)](/images/{{page.url}}_7.png){: target="_blank"}

<pre>

    ✎ 날짜가 하루라도 중복되거나 이전 제출 완료된 파일을 재 전송하는 경우 
           이전에 제출된 자료는 모두 삭제됨
</pre>
[![](/images/{{page.url}}_8.png)](/images/{{page.url}}_8.png){: target="_blank"}

<pre>
    ✎ 자료전송은 <red>2021년 1월 7일 까지 제출</red> 가능
    ✎ 기간별 구분하여 수시로 제출 가능
    ✎ <red>단, 기간이 하루라도 중복되면 가장 마지막에 제출한 자료만 인정</red>
    	    예 : 2020.01.01 ~ 2020.03.31  100건 제출 후
                 2020.03.31 ~ 2020.05.31  50건 다시 제출한 경우
                 실제 국세청에는 50건 만 제출된 것으로 인정됨<red> (3월 31일이 중복되었음)</red>
    ✎ 타사 프로그램을 사용하다 NixPen으로 변경하여 사용하는 경우
             * 타사 프로그램 회사에 문의하여 타사프로그램으로 사용한 기간별 자료를 제출 하시고
                NixPen을 사용하기 시작한 시점부터는 기간이 중복되지 않도록 기간을 설정하여 
                파일 생성을 하시고 자료를 송신 하시면 됩니다.

    <red>※ 국세청 자료전송 관련 문의 사항은 국세청 연말정산간소화 고객센터에 문의 바랍니다.</red>
           <blue>문의처 : 국번없이 126</blue>
</pre>
